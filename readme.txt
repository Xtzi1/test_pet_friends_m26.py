Тесты объеденены в класс TestPetFriends.

Тесты разбиты на несколько групп, группы описаны в файле pytest.ini.

Тесты были переработы так, чтобы они использовали фикстуру получения ключа API, а не делали это каждый раз внутри себя.

Несколько тестов используют одни и те же фикстуры setup и teardown.

В файл api добавлен декоратор, который логирует запросы в API тестах и применен ко всем запросам в файле api.py.
